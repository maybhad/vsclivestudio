Implementation Notes

The two most useful values to watch are:

session.rpcs[uuid].buffer = requested base buffer for that peer.

stats.chunked_mode_video.buffer_buffer = live effective target after any adaptive logic.

If you want deterministic QA, disable or stub the adaptive buffer helper in a dedicated test harness. For real integrations, leave adaptation on and observe both the requested base and the effective target.
